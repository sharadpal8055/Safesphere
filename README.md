# 🛡️ SafeSphere — AI Disaster Risk Prediction Platform

> AI-powered disaster intelligence system that predicts potential disaster risks using machine learning, environmental data analysis, and intelligent prediction workflows.

SafeSphere is an AI-driven disaster risk analysis platform designed to estimate disaster possibilities by analyzing environmental and historical datasets.  
The system applies machine learning techniques to identify risk patterns and provide intelligent predictions that can support early awareness and decision-making.

---

##  Overview

Natural disasters cause severe damage due to delayed prediction and lack of early insights.

SafeSphere focuses on:

- Predicting disaster risk levels
- Analyzing environmental factors
- Processing historical disaster data
- Generating intelligent risk insights
- Supporting proactive decision making

The platform combines data preprocessing, machine learning models, and interactive visualization to make disaster analysis more accessible.

# ⚙️ Installation & Setup


## 1. Clone Repository


```bash
git clone https://github.com/your-username/SafeSphere.git
```


Navigate:

```bash
cd SafeSphere
```


---

## 2. Create Virtual Environment


Windows:

```bash
python -m venv venv

venv\Scripts\activate
```


Linux / Mac:

```bash
python3 -m venv venv

source venv/bin/activate
```


---

## 3. Install Dependencies


```bash
pip install -r requirements.txt
```


---

## 4. Run Application


```bash
streamlit run app.py
```


Open:

```text
http://localhost:8501
```
